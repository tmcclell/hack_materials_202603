What's Next: Copilot SDK & OpenClaw

⏱ 5 minutes
🎯 Key Message
"You can embed Copilot's brain into your own apps — and OpenClaw shows what that looks like."

Copilot SDK

  • @github/copilot-sdk is in technical preview
  • Available for Node.js, Python, Go, .NET
  • Capabilities: multi-turn conversations, tool orchestration, model routing, streaming
  • Build custom agents, add tools, integrate into your own workflows
  • Authentication and model management handled out-of-the-box

OpenClaw

  • A real-world example of building on Copilot — open-source AI assistant framework
  • Uses Copilot as a model provider
  • Shows the pattern: your app → SDK → Copilot models
