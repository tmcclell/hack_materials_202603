Squad: AI Agent Teams Alpha

⏱ Referenced from Autonomous Copilot section
🎯 Key Message
"Squad gives you an AI development team — each member runs in its own context, shares decisions, and persists across sessions."

What is Squad?

Open-source project by Brady Gaster — github.com/bradygaster/squad

  • "AI agent teams for any project. One command. A team that grows with your code."
  • Each team member (frontend, backend, tester, lead) runs in its own context
  • Parallel execution — multiple agents work simultaneously
  • Session persistence — crashes resume from checkpoint
  • Decision logging — every decision recorded in .squad/decisions.md

How It Works

  • npm install -g @bradygaster/squad-cli
  • squad init → creates .squad/team.md
  • Talk to agents: @Keaton, analyze the architecture or Build the login page
  • Coordinator routes to right agents

Pattern Comparison

Pattern Agents Mode Style
Single Copilot Agent One One task Interactive
Autopilot One Autonomous End-to-end
Squad Multiple specialized Coordinated Persistent

⚠️ Alpha Software

  • Experimental — APIs may change
  • Great for exploration and prototyping, not production dependencies
