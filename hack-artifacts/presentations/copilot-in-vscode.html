How Copilot Works Inside VS Code

Understanding the three interaction modes, context system, and model selection

1 The Three Interaction Modes

⌨️ Inline Completions Fastest

Ghost text appears as you type — just press Tab to accept.

  • Suggests whole lines or blocks of code
  • Context drawn from open files and nearby comments
  • Multi-line suggestions for repetitive patterns
  • Press Esc to dismiss, Alt+] to cycle alternatives

💬 Copilot Chat Panel Conversational

Ask questions, get explanations, and request code changes in natural language.

  • /explain — explain selected code
  • /fix — fix bugs in selection
  • /tests — generate unit tests
  • /doc — add documentation
  • @workspace — ask project-wide questions
  • Multi-turn conversations with context memory

🤖 Agent Mode Autonomous

Copilot plans, executes, and self-corrects across your entire project.

  • Multi-file edits in one session
  • Runs terminal commands (build, test, lint)
  • Iterative loop: Plan → Execute → Verify → Self-correct
  • Reviews its own output and fixes errors
  • Toggle via the Agent Mode button in Chat

What feeds the context window?

SourceHow It Works
Currently open fileHighest priority — code near your cursor is always included

💡 Better context = better output. The more relevant code Copilot can see, the more accurate and useful its suggestions. Use #file and @workspace liberally!

3 Model Selection

Available Models in VS Code

Model Provider Best For Speed
Claude Sonnet 4.x Anthropic Great all-rounder, excellent code quality Fast
Claude Opus 4.5 / 4.6 Anthropic Complex reasoning, architecture decisions Medium
GPT-5.x OpenAI Versatile, strong at many tasks Fast
GPT-5 mini OpenAI Quick tasks, included model (free) Very Fast
Gemini 3 Pro Google Large context window, multimodal Fast

🔄 Auto Selection

Select "Auto" in the model picker and Copilot chooses the best model for each request. You also get a 10% discount on premium request costs.

🎯 Manual Selection

Click the model picker dropdown at the bottom of the Chat panel to manually choose a specific model for specialized tasks.
