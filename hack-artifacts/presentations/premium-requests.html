Premium Requests & Model Selection

Understanding costs, quotas, and how to get the most from your Copilot plan

1 What Are Premium Requests?

💡 The Basics

Every Copilot interaction — a chat message, an inline suggestion with a premium model, an agent action — is a "request."

  • Basic requests with included models are FREE on paid plans — no quota consumed
  • Advanced models and features consume premium requests from your monthly quota
  • Each plan gets a fixed number of premium requests per month

4 Auto Model Selection = 10% Discount

🎯 "Auto" is the Smart Default

When you select "Auto" in the VS Code model picker, Copilot automatically chooses the best available model for each request — and gives you a 10% discount on every premium request.

✅ Auto WILL

  • Pick the best model for the task
  • Apply 0.9× multiplier to all 1× models
  • Respect org policy restrictions

❌ Auto WON'T

  • Use models with multipliers >1×
  • Use models blocked by your org
  • Override explicit manual selection

6 What Happens When You Run Out?

🔋

You're not locked out — you're downgraded

When your premium request quota hits zero:

  • Included models still work — GPT-5 mini, GPT-4.1, and GPT-4o remain fully available at no cost
  • Premium models become unavailable until your quota resets on the 1st
  • Inline completions continue using included models
💳 Overage billing (org admins)
Admins can enable overage at $0.04/request so users are never blocked.
🛑 Hard budget caps
Alternatively, set a budget that blocks usage when reached — no surprise bills.
